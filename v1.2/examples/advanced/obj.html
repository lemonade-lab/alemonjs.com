客户端 Client ​

用于直接调用接口的客户端对象

让开发框架不会因为兼容性导致接口缺失

同时能保持原生平台特有功能

ClientFile ​

ts
import { ClientFile } from 'alemonjs'
1

ts

  '/plugins/point/public/img/ionc.png'
)
