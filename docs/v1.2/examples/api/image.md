# 图片制作

:::tip 注意

该模块设计已独立。
请阅读[react-puppeteer](https://github.com/lemonade-lab/react-puppeteer)

:::


确保 ts 开启了 jsx

- tsconfig.json

```json
{
  "compilerOptions": {
    "jsx": "react",
    "noEmit": true,
    "allowImportingTsExtensions": true
  }
}
```

安装 react

- npm

```sh
npm install react  react-dom @types/react-dom -D
```
